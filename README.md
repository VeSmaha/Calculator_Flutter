# Calculator_Flutter
 Calculadora básica com dart e flutter
 
## Pré-requisitos

- Flutter SDK instalado
- Dart SDK instalado

## Instalação

1. Clone este repositório em sua máquina local:
2. flutter packages get (obtem dependencias)
3. flutter run
